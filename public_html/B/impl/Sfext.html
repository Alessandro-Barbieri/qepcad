Q E P C A D - SFEXT Extensions to QEPCAD Documentation

This package contains some of the solution formula extensions described in my PhD thesis. The entry-point to these routines from QEPCAD is currently the function 
void SOLEXTINT();
defined in sfcons/SOLEXTINT.c.

The main datastructures used in these packages are described in these pages:

  • Formula
  • Polynomial Index
  • PCAD
  • RCAD
  • ESPCAD
